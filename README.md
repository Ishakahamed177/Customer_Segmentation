## 📊 Customer Segmentation Analysis
Customer segmentation analysis using python K-Means clustering at Power BI

## 📌 Project Overview

The Customer Segmentation Analysis project focuses on identifying distinct groups of customers based on their demographic characteristics and spending behavior. By applying Machine Learning techniques, customers are segmented into meaningful clusters that help businesses understand customer preferences and design targeted marketing strategies.

## 🎯 Objective

To segment customers into different groups using K-Means Clustering and analyze their purchasing behavior based on factors such as age, gender, education, marital status, annual income, and spending score.

## 🛠️ Tools & Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Power BI
